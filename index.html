Software Release Notes

v2 hardware products: v2.5.0 -- Latest Release

  • New: Language support is added for the web interface: German, Dutch, Swedish, Norwegian, English, Turkish, Danish, Finnish.
  • New: Help section with more information how to use the system.
  • New: Identify button is added for each luminarie in commissioning mode.
  • New: LED status blinking is limited to one hour post-boot for all non-commissioned units.
  • New: Memory improvements for the first-time commissioning
  • New: Login page lockout timer is added for throttling.
  • Improvement: Streamlined the embedded web interface for better usability and cross-device compatibility.
  • Security and stability updates.
  • Mobile App: Hardware and firmware compatibility is verified before applying an OTA update, preventing mismatched upgrades.
  • Mobile App: Firmware Update section has been moved to the main page.
  • Mobile App: A new button 'Projects' has been added to the main page to search for all projects on the WiFi network.
  • Mobile App: New language options have been added.

v2.1.2

  • Fixed web server disappearing after a prolong usage of the system
  • Added the ability to fetch emergency lighting reports from the local web interface
  • Fixed system stability causing too dynamic reorganization of luminaries
  • Improved OTA reliability significantly
  • Cloud improvements for saving emergency lighting reports for up to 3 years
  • The email feature is disabled until confirmed via the activation link
  • Security updates
  • Performance enchancements

v2.0.2

  • Added Firmware Version column to monitoring page
  • Added Signal Strength (RSSI) column to monitoring page (default -120 if unknown)

v2.0.1

  • Web interface redesign
  • New feature: Project Information section (Project Name, IP Address, Device ID, Total Devices)
